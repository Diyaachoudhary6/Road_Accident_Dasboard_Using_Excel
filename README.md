# 🚦 Road Accident Analysis Dashboard | Excel Data Analytics Project

## Project Description / Purpose
This project analyzes road accident data and transforms raw records into meaningful business insights using Microsoft Excel. The dashboard provides a comprehensive view of accident casualties by severity, vehicle type, road type, road surface conditions, light conditions, and geographical areas.
The objective of this dashboard is to help stakeholders identify accident trends, high-risk factors, and key patterns that can support road safety initiatives and policy decisions.

## Business Questions Answered
- What is the total number of casualties?
- How many casualties are classified as Fatal, Serious, and Slight?
- Which vehicle types are involved in the highest number of casualties?
- Which road types contribute most to accidents?
- How do road surface conditions impact casualties?
- Are accidents more frequent during daylight or darkness?
- What is the distribution of casualties across Urban and Rural areas?
- How do monthly casualty trends compare with previous years?

## Tech Stack
- Microsoft Excel
- Power Query (Data Cleaning & Transformation)
- Pivot Tables
- Pivot Charts
- Slicers & Timelines
- Dashboard Design & Data Visualization
- Advanced Excel Functions

## Key Insights 📈
- Slight casualties account for the majority of total casualties.
- Cars contribute the highest number of accident casualties.
- Single carriageways experience the largest share of accidents.
- Most casualties occur in urban areas.
- Daylight conditions record more casualties than darkness.
- Road surface conditions significantly influence accident outcomes.
